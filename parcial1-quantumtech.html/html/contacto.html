Preguntas Frecuentes (FAQ)

¿Qué lenguajes usa este sitio?

Usamos HTML5 para la estructura, CSS3 para el diseño y **JavaScript** para la validación del formulario.

¿Dónde se guardan los datos?

Los datos se envían a un script de servidor (`guardar.php`) para su almacenamiento en un archivo de datos, cumpliendo con el requisito del parcial.
